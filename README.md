# SMSBomber

## Hakkında

Bu proje Python ile geliştirilmiş bir test uygulamasıdır.

## Özellikler

* Hızlı kullanım
* Komut satırı desteği
* Açık kaynak

## Gereksinimler

* Python 3.10+
* venv

## Kurulum

```bash
git clone https://github.com/killoed/Scorpionsmsbomber.git
cd SMSBomber
bash scorpionsmsbomber.sh
```

## Çalıştırma

```bash
bash scorpionsmsbomber.sh
```

## Uyarı

Bu proje yalnızca eğitim, geliştirme ve izinli test amaçları için kullanılmalıdır. Yetkisiz kullanımdan kullanıcı sorumludur.
